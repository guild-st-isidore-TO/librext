## LibRext Type Scale

### Type Scale Values

This type scale system adds a set of **Display** levels. These are similar to the typical **Heading** levels, but extend the system into larger font sizes.

| Scale level | Name | Description |
| --- | --- | --- |
| 1 | **tiny** | Tiny |
| 2 | **small** | Small |
| 3 | **body** | Base font size |
| 4 | **h6** | Heading 6 |
| 5 | **h5** | Heading 5 |
| 6 | **h4** | Heading 4 |
| 7 | **h3** | Heading 3 |
| 8 | **h2** | Heading 2 |
| 9 | **h1** | Heading 1 |
| 10 | **d6** | Display 6 _(title)_ |
| 11 | **d5** | Display 5 _(headline)_ |
| 12 | **d4** | Display 4  |
| 13 | **d3** | Display 3 |
| 14 | **d2** | Display 2 |
| 15 | **d1** | Display 1 |
