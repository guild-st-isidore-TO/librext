As for shapes, LibRext deals only with boxes. There are several options available for border radius, box shadow, widget heights, and card styles

Border Radius — A larger radius fits with larger boxes

Box Shadow — Larger sizes imply more distance between box layer and parent layer

Widget Heights — Presets for common/consistent UX elements. Like inputs, buttons, chips, etc.
