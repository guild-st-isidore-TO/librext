Typography

Back to Specimens

LibRext has a heavy focus on old-style serif fonts, as it looks to early Renaissance printed works as inspiration. The design system’s text styles are organized by "Font Roles" and "Document Roles”

Font Roles are functional roles that can be shared by multiple elements.

Document Roles correspond to common document styles and HTML/Markdown structure

Typefaces

LibRext text styles revolve around six typefaces:

  • Cinzel — Used for display
  • Cormorant Garamond — Used for headings
  • Crimson Text — Used for body, headings (alt)
  • Open Sans — Used for labels, UI headings
  • Lato — Used for UI body
  • Inconsolata — Used for monospace, code

Font Roles

Font definitions available for the design system. These are defined by general roles, and can be used by several (more specific) styles.

Document Roles

Font definitions for text within a document. These are defined by specific roles in a page or screen. Such as titles, subtitles, headings, etc.
